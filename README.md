## Run Locally

Clone the project

```bash
  git clone https://github.com/gonzalezvictorjuan/socketio-chat.git
```

Go to the project directory

```bash
  cd socketio-chat
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

or Start the server in mode dev

```bash
  npm run dev
```

Open two browser tabs with http://localhost:3000/
and chat!
